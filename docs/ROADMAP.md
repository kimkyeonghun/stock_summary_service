# BriefAlpha Roadmap

## M4-FUTURE-01: 요약 생성 모델 상향
- 목적: item/digest/report 생성 품질 향상
- 범위: 상위 모델 후보 A/B 테스트, 비용/지연 비교, 자동 롤백 기준 정의
- 산출물: 모델별 품질 점수표, 운영 기본 모델 결정안

## M4-FUTURE-02: 번역 품질 자동평가
- 목적: 번역 품질과 사실 보존률 상시 모니터링
- 범위: 숫자/날짜/퍼센트/티커 보존율 자동 측정, 샘플 휴먼 검수 루프
- 산출물: 일별 품질 리포트, 임계치 미달 시 경고 알림

## M4-FUTURE-03: KR/US 모델 분리 전략
- 목적: 시장별 언어/문체 특성에 맞는 요약·번역 품질 최적화
- 범위: KR/US 별 생성 모델, 번역 모델, 프롬프트 정책 분리
- 산출물: 시장별 모델 라우팅 룰, 비용 예산 정책, 장애 대응 플랜
